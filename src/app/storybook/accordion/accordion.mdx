### When to use

#### Accordion Types
- Use single accordion for simple expandable content
- Use multiple accordion for grouped related content
- Use nested accordion for hierarchical information

#### Accordion Features
- Use with icons to indicate expandable state
- Use with custom headers for better context
- Use with different sizes based on content
- Use with different styles to match your design

#### Content Organization
- Use for FAQs and help sections
- Use for complex form sections
- Use for nested navigation
- Use for collapsible content sections

#### Best Practices
- Keep content concise and focused
- Use clear and descriptive headers
- Consider mobile responsiveness
- Maintain consistent styling
