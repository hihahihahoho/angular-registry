### When to use

#### Alert Types
- Use success alerts for positive feedback
- Use error alerts for error messages
- Use warning alerts for important notices
- Use info alerts for general information

#### Alert Features
- Use with icons to enhance message meaning
- Use with close button for dismissible alerts
- Use with different sizes based on importance
- Use with different styles to match your design

#### Message Types
- Use for form validation feedback
- Use for system status updates
- Use for important announcements
- Use for user action confirmations

#### Best Practices
- Keep messages clear and concise
- Use appropriate alert type
- Consider alert placement
- Make important alerts dismissible
- Use consistent styling
